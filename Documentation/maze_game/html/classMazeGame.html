Detailed Description

Author
Matthew Atkins

This is main ROS node and performs the majority of the interactions between the MazeGame code and ROS itself.
It provides the following services:

  • add_robot: adds a robot with the name given in the service request.
  • remove_robot: removes the robot with the name given in the service request.
  • start_robot: sends the robot to the start of the maze and sets its state to MOVING_TO_START click here for more info.
  • restart_robot: resets the robot with the given name back to it's initialization values and set it's state to IDLE
  • request_maze_markers: causes the maze's point to be published as visualization_msgs/Marker message on the /maze_game/markers topics
  • request_average_scores:
  • submit_scores:
  • export_run_data: exports the data for the robot with the given name or all available data if the name given is 'ALL'. The data is stored as a CSV file in the packages 'export' directory. Actual export is done here
